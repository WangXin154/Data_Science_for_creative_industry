📊 Summary of project content

1. Research objectives

- Analyze whether the audio characteristics of Spotify music (danceability, energy, rhythm, instrumentality, loudness, etc.) change over time from 2015 to 2025
- Use linear models to capture long-term trends
- Explore the predictive power of audio features on song popularity

2. Data processing

- Dataset: 85,000 songs, 19 variables
- Missing values are handled (only 21 tracknames are missing, accounting for 0.02%)
- Feature engineering: extract year, perform log10 transformation on streamcount

3. Main analysis

Time trend analysis:
- Linear regression using scipy.stats.linregress()
- Most audio features were found to be very stable across ten years, with no significant linear trends
- p-value > 0.05 (not significant) for all features

Genre group analysis:
- Analysis of Top 5 genres (Metal, Jazz, Hip-Hop, Classical, Rock)
- It was found that different genres have different trends, and the overall average masks the differences between genres

Popularity prediction model:
- Baseline model: popularity ~ log10(streamcount), R² ≈ 0.546
- Single feature model: Each audio feature alone has very weak explanatory power on popularity (R² ≈ 0)
- Multiple linear regression: including all audio features logstreamcount, R² ≈ 0.546
- Conclusion: Play volume is the main predictor of popularity, with audio features making little independent contribution
